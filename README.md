# gRPC를 활용한 파일 송수신 프로그램

### Clone and Build this Repository

## **Try it!**

### SERVER

- ./remote_server  <DATASET_FOLDER_PATH>

```bash
./remote_server ../dataset/
```

### CLIENT

- ./remote_client <DOWNLOAD_FOLDER_PATH>
```bash
mkdir ../download
./remote_client ../download/
```
